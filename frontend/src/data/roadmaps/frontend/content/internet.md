# Internet

The internet is a global network of interconnected computers that communicate using standardized protocols. Understanding how it works is foundational to frontend development.

## How does it work?

When you type a URL, your browser:

1. Performs a **DNS lookup** to convert the domain name to an IP address
2. Initiates a **TCP connection** (handshake) with the server
3. Sends an **HTTP/HTTPS request**
4. The server responds with the content (HTML, CSS, JS, etc.)
5. The browser parses and renders the page

## Key Concepts

- **IP Address** — Unique identifier for every device on the network
- **TCP/IP** — The fundamental communication protocols of the internet
- **Packets** — Data is split into small packets for transmission
- **Bandwidth** — The amount of data that can be transferred per second
- **Latency** — The time it takes for a packet to travel from source to destination
