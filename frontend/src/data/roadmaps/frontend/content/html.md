# HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure and meaning of web content.

## What you'll learn

HTML is not a programming language — it's a **markup language** that defines the structure of your content. HTML consists of a series of **elements** which tell the browser how to display content.

## Core Concepts

### Elements & Tags
```html
<p>This is a paragraph.</p>
<a href="https://example.com">This is a link</a>
<img src="image.jpg" alt="Description" />
```

### Document Structure
```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>Page Title</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>
```

### Semantic Elements
Use semantic tags to give meaning to your HTML:
- `<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<aside>`, `<footer>`
- `<h1>` to `<h6>` for headings (one `<h1>` per page)

## Attributes
Elements can have **attributes** that provide additional information:
- `id` — Unique identifier
- `class` — CSS class names
- `href` — Link destination
- `src` — Resource source
- `alt` — Alternative text for accessibility
